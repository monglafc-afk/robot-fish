# robot-fish

> 用来开发和测试一款视频会议软件。

## 项目简介

robot-fish 是一个致力于提升远程沟通与协作的视频会议软件项目，包含基本音视频会议、屏幕共享、实时聊天等功能。

## 功能特性

- 多人在线视频会议
- 实时音视频同步
- 屏幕共享
- 文字即时聊天
- 会议录制
- 用户管理与权限控制

## 项目结构

```
robot-fish/
├── docs/             # 文档
├── src/              # 源码
│   ├── backend/      # 后端服务
│   └── frontend/     # 前端页面
├── tests/            # 测试
├── .gitignore        # Git忽略文件
├── package.json      # 项目依赖配置 (如使用 Node.js)
├── README.md         # 项目说明
```

## 安装与运行

```bash
git clone https://github.com/monglafc-afk/robot-fish.git
cd robot-fish
# 按照 src/backend 或 src/frontend 下指引启动
```
